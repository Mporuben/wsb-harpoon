# Folders
After succesful instalation Harpoon will create `wsb-harpoon` folder in your home directory 


```
wsn-harpoon
├── /.internal
│   |── /plugins  
│   └── /data
├── /raw
│   └── [ticker]
│       └── 10k_[year].xlsx
└── /exports
    └── [ticker].xlsx
```

# raw
place for raw reports

# exports
place for exported reports

# .internal
`.internal` folder is used for managing plugins and internal data.
